Low Power Full Adder Using GDI XOR and PTL-Based MUX

Designed and simulated a low-power 1-bit Full Adder using GDI-based XOR gates and a Pass Transistor Logic (PTL) multiplexer in Cadence Virtuoso. The proposed architecture reduces power consumption, transistor count, area, and propagation delay compared to conventional CMOS designs. Transient analysis verifies correct Sum and Carry operation for different input combinations. The design achieves an average power consumption of 154.9 µW, making it suitable for energy-efficient VLSI and high-speed arithmetic applications.

Features
GDI XOR implementation
PTL-based Carry generation
Reduced power, area, and delay
Compact transistor-level design
Cadence Virtuoso simulation
Applications
Low-power VLSI systems
Arithmetic circuits
ALU design
Portable electronic devices
